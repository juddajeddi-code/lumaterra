# Lumaterra.education

*Repository:* Public repository for the Lumaterra.education learning page.  

*Purpose:*  
This is a *non-commercial educational platform* dedicated to:  
- Botanical literacy  
- Cannabinoid science and understanding  
- Regulatory-aware formulation education  

*Contents:*  
- index.html — main HTML page  
- assets/ — folder containing images, logos, OG previews, and optional CSS/fonts  
- .gitignore — ignores temporary, OS, and editor files  

*Deployment:*  
The site is designed to be deployed via *Cloudflare Pages* as a static HTML/CSS website.
